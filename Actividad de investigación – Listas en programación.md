* ¿Qué es una lista en programación?
  Una lista es una estructura de datos que permite almacenar una colección de elementos de forma ordenada.


* ¿Para qué se utilizan las listas? Dar un ejemplo de un problema donde sea útil utilizar una lista.
  Se utilizan cuando no sabes de antemano cuántos datos vas a recibir o cuando necesitas manipular un grupo de elementos
  relacionados.
  Un ejemplo sería, a la hora de programar un carrito de compras donde necesitas almacenar los productos que se han
  seleccionado.


* ¿Qué diferencia hay entre guardar un solo valor en una variable y guardar varios valores en una lista?
  La diferencia es que una variable solo puede almacenar un valor, mientras que una lista puede almacenar varios
  valores.


* ¿Qué operaciones básicas se pueden realizar con una lista? (por ejemplo: agregar, eliminar, recorrer, etc.).
  Las operaciones son: add, remove, get, set, con un for/while recorremos la lista.
